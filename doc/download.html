Access to the FTP Directory

Some people have problems getting to ftp.fox-toolkit.org via FTP. This is due to FTP's funny way of setting up TWO connections, one for data and one for commands.
It is a problem because the data connection comes in at a random (high) port number, and may therefore be blocked by the firewall.
You can try "passive" mode FTP (typically "ftp -p") but if that doesn't work your best bet is to just try to access the directory via HTTP. 

Via FTP:  ftp://ftp.fox-toolkit.org/pub.


Via HTTP: http://www.fox-toolkit.org/ftp/.
